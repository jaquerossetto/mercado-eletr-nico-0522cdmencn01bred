# Aula 12 - Coleção: Array, ArrayList e List

## Resumo do conteúdo da aula

### Sintaxe

#### Array: Vetor unidimensional

```cs

String[] Frutas;
Frutas = new String[5];

Frutas[0] = "Abacaxi";
Frutas[1] = "Pêra";
Frutas[2] = "Maçã";
Frutas[3] = "Banana";
Frutas[4] = "Mamão";

```

#### Array: Vetor multidimensional

```cs

String[,] Coordenada;
Coordenada = new String[2,2];

Coordenada[0,0] = "⭐";
Coordenada[0,1] = "2";
Coordenada[1,0] = "3";
Coordenada[1,1] = "4";

```

#### List: Lista de valores

```cs

List<String> Frutas;
Frutas = new List<String>();

Frutas[0] = "Abacaxi";
Frutas[1] = "Pêra";
Frutas[2] = "Maçã";
Frutas[3] = "Banana";
...

```

#### ArrayList: Coleção de valores

```cs

ArrayList Gaveta;
Gaveta = new ArrayList();

Gaveta[0] = "string";
Gaveta[1] = null;
Gaveta[2] = 1;
Gaveta[3] = 2.5;
Gaveta[4] = true;

```
